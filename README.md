### üîµ Option 1: Virtual Environment (Recommended)
**[EN]** Run with isolated environment (Prevents library conflicts).
1. **Clone:** `git clone git@github.com:NKSR22/AI-Hand-Gesture-Recognition.git`
2. **Setup venv:** 
   - Windows: `python -m venv venv` -> `.\venv\Scripts\activate`
   - Mac/Linux: `python3 -m venv venv` -> `source venv/bin/activate`
3. **Install:** `pip install -r requirements.txt`
4. **Run:** `python main.py`
